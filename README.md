#### Overview

This application provides a banking solution for both bank members and administrators. It allows members to submit reimbursement requests and administrators to manage and approve them.

#### Usage

1. Access the application through the provided URL.
2. Bank members can navigate to the "Requests" page to view and submit reimbursement requests.
3. Bank administrators can access the "Management" page to review and approve/deny reimbursement requests.

#### Features

- Intuitive Sidebar Navigation: Easily navigate between different sections of the application.
- Responsive User Interface: Accessible on various devices for seamless user experience.
- Secure Authentication: Ensure user data privacy and protection through secure authentication mechanisms.

#### Disclaimer

This application is for demonstration purposes only and does not handle real financial transactions. Any data entered or submitted is for testing and evaluation purposes only.
